# Rust Practice

Rustの学習・練習用リポジトリです。

## 環境構築

このリポジトリはGitHub Codespacesで開発環境が自動的にセットアップされます。

### Codespacesでの起動方法

1. 緑色の「Code」ボタンをクリック
2. 「Codespaces」タブを選択
3. 「Create codespace on main」をクリック

起動後、自動的にRust環境がインストールされます。

## 学習の進め方

### 1. Hello World

```bash
cargo new hello_world
cd hello_world
cargo run
```

### 2. 変数と型

```bash
cargo new variables
cd variables
# src/main.rsを編集
cargo run
```

### 3. 関数

```bash
cargo new functions
cd functions
# src/main.rsを編集
cargo run
```
